# Exercícios React e JavaScript

Esta pasta contém uma coleção de exercícios e desafios para prática em React e JavaScript.

## Estrutura

- `desafio/`: Desafios mais complexos.
- `ex001/` a `ex06/`: Exercícios básicos a avançados de React.
- `js/`: Exercícios focados em JavaScript puro.

Cada pasta tem seu próprio README com instruções.
